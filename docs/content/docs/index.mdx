# Generate Metadata

Generate Metadata is a TypeScript library that provides AI-powered, dynamic SEO metadata generation for web applications. It fetches optimized metadata from a remote API service and integrates seamlessly with popular frameworks like Next.js and TanStack Start.

## Features

- **AI-Powered**: Generates optimized titles, descriptions, and social tags
- **Framework Support**: Next.js and TanStack Start adapters
- **Type Safe**: Full TypeScript support
- **Smart Caching**: Built-in caching with graceful fallbacks
- **Development Mode**: Works offline with fallback metadata
- **Social Ready**: OpenGraph and Twitter Card support

## How It Works

1. **Initialize**: Create a client with your project's DSN (Data Source Name)
2. **Generate**: Call metadata functions with your page path and configuration
3. **Optimize**: The AI service analyzes your content and generates optimized metadata
4. **Render**: Framework-specific adapters format the metadata for your application

```ts
import { GenerateMetadataClient } from "generate-metadata/next";

const client = new GenerateMetadataClient({
  dsn: process.env.NEXT_PUBLIC_GENERATE_METADATA_DSN,
});

export const generateMetadata = client.getMetadata(() => ({
  path: "/",
  fallback: {
    title: "My App",
    description: "Welcome to my application",
  },
}));
```
